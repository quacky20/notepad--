HOW TO RUN:
1. Download all the files from the download folder in this repository.
2. Put the main.exe file and icon.ico file in the same directory.
3. Right click on the main.exe file and go to Send to and select Desktop (create shortcut).
4. Go to desktop and rename the shortcut.
5. You can also directly open the application from the directory where both the main.exe and icon.ico are stored by opening the main.exe executable file.
